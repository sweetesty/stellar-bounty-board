Example: Bounty Detail Page Meta Tags

This is an example of what the HTML head will look like when viewing a bounty detail page.

Social Media Preview

When this page is shared on social media, platforms will display:

  • Title: Add authentication feature
  • Description: Implement OAuth2 authentication for the API • Reward: 500 XLM
  • Image: Repository owner's GitHub avatar
  • URL: https://stellar-bounty.com/bounties/bounty-123

How It Works

  1. User navigates to a bounty detail page (e.g., /bounties/bounty-123)
  2. React loads the BountyDetailPage component
  3. When bounty data is fetched, the useEffect hook calls updateSocialMetaTags()
  4. The function dynamically updates all meta tags in the document head
  5. Social media crawlers read these tags to generate rich previews

Default Meta Tags (Homepage)

When not viewing a bounty detail page, the default meta tags are:

  • Title: Stellar Bounty Board
  • Description: Fund GitHub issues with on-chain style escrow
  • Type: website
  • No image (uses platform defaults)
